        \section{Challenge Tasks}

            This section describes how I implemented the challenge tasks and what I did in addition.

            \begin{itemize}[leftmargin=*]
                \item \textit{``Add characters to your game.''}
                
                    I did stuff to do this trolololol

                \item \textit{``Extend the parser.''}
                
                    I entirely replaced how the parser worked, I began by implementing a basic model of \codeword{Command}, it took a few iterations but I settled on providing Regex \codeword{Pattern}s.

                    This approach has several benefits:

                    \begin{itemize}
                        \item Powerful Regex at low performance cost due to the low number of commands.
                        \item Ability to have named capture groups which are then interpreted as arguments.
                    \end{itemize}

                    For each known \codeword{Command}, I would create a \codeword{Matcher} for each \codeword{Pattern}, execute it against the arbitrary command from the user and then pass it into a wrapper class \codeword{Arguments} which lets me safely pull out named groups.
                
                \item \textit{``Add a magic transporter room.''}
                
                    To do this, I added a new \codeword{RoomWormHole} which I made implement \codeword{EventEntityEnteredRoom} to listen for when any entities entered the room, as soon as one is detected, a short animation is begun and a room is selected at random to teleport the user to. Allowing the user to go to \textbf{any room} may interfere with my story so I chose to only spawn the user at any outside areas of the map.
